# AI Chat Assistant

一个基于 Next.js 和 OpenAI API 构建的智能对话助手。

## 功能特点

- 现代化的用户界面
- 实时对话功能
- 支持暗色模式
- 响应式设计
- OpenAI API 集成

## 技术栈

- Next.js 13+
- TypeScript
- Tailwind CSS
- Framer Motion
- OpenAI API

## 开始使用

1. 克隆仓库：
```bash
git clone https://github.com/liupengstudy/jack-chat-assistant.git
```

2. 安装依赖：
```bash
npm install
```

3. 配置环境变量：
创建 `.env.local` 文件并添加：
```
OPENAI_API_KEY=your_api_key_here
```

4. 启动开发服务器：
```bash
npm run dev
```
